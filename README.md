# 项目简介

## 核心功能
- 支持图片、视频及摄像头进行车牌区域检测
- 支持选择不同模型进行检测
- 界面实时显示目标位置、目标总数及置信度信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 支持自定义置信度和IOU阈值
- 支持选择是否显示标签和原图
- 提供单张图片、图片文件夹、视频文件和本地摄像头四种识别方式

## 适用场景
- 车牌识别系统开发与研究
- 智能交通监控系统
- 车辆管理平台
- 安防监控系统
- 停车场管理系统
- 车牌数据分析与处理

## 技术特点
- 基于YOLOv8目标检测算法
- 采用Python3.9开发环境
- 集成opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供完整的UI界面，支持交互操作
- 包含8493张标注数据集，专为中国车牌检测优化
- 提供预训练模型，配置成功环境可直接使用
